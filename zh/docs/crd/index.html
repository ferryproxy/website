    CRD #

    Hub 定义一个 Hub, 是与其他 Hub 通信地址

    Route 定义 Hub 之间的单个 Service 的路由规则

    RoutePolicy 是用于定义通用的复合型的路由规则, 最终会转换成 Route

    ServiceImport/ServiceExport 是为了兼容 mcs api, 最终会转换成 RoutePolicy

    TunnelRule 是建立隧道的规则, 还未定义CRD, 使用 ConfigMap 存的

    ServicePort 是为导入的隧道建立对应的 Service/Endpoint, 还未定义CRD, 使用 ConfigMap 存的
